# Basic Syntax (Paste into editor to see it in use)

## Headings
Use `#` for headings. Example:

# H1
## H2
### H3

## Bold and Italics
Use **bold** or __bold__; *italic* or _italic_. Example:

**bold** _italic_

## Lists
Unordered:
- Item 1
- Item 2
- Item 3

Ordered:
1. First
2. Second
3. Third

## Links and Images
[Link text](https://example.com)
![Alt text](image.jpg)

## Code
Inline: `code`
Block:
```javascript
console.log('hello');
```

## Blockquotes
> This is a blockquote.

---
